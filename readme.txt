Cara mengeksekusi program:
1. Buka command prompt, ganti direktori ke folder bin: ~/Algeo-16015/bin
2. Pada commandline, ketikkan "python main.py"

Cara memasukkan input pada program:
1. Saat pertama dibuka, program akan meminta jumlah titik sudut yang akan digambar. Masukan titik lebih dari 2.
2. Masukan titik harus berurutan searah jarum jam atau sebaliknya, tidak disarankan memasukkan titik sudut tidak berurutan.
3. Setelah gambar muncul pada jendela OpenGL, program akan meminta input command pada jendela command prompt.
4. Masukan command sesuai yang tertulis pada spesifikasi Tugas Besar (ditampilkan pada menu)
